# Clinic Patient Billing System

A C++ billing system for a clinic managing patients across 7 days.

## Features
- Supports consultation and medical test services (Blood Test, MRI, CT Scan, X-Ray)
- Handles multiple services per patient in one visit
- Automatically applies 10% discount for bills above 2000
- Automatically applies 15% discount for bills above 3000
- Runs for 3 patients per day across a full week

## How to Run
1. Compile: g++ Clinic_Patient_Billing_System.cpp -o clinic
2. Run: ./clinic

## Tech Used
C++ — nested loops, do-while loops, conditionals
